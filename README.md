# Dinesh Karuppasamy - Portfolio

A modern, responsive portfolio website built with React and Tailwind CSS.

## 🚀 Features

- **Modern Design**: Clean and professional design with smooth animations
- **Responsive**: Fully responsive design that works on all devices
- **Fast**: Built with Vite for optimal performance
- **SEO Friendly**: Optimized for search engines
- **Easy to Deploy**: Ready for GitHub Pages deployment

## 🛠️ Technologies Used

- React 18
- Tailwind CSS
- Framer Motion (animations)
- Lucide React (icons)
- Vite (build tool)

## 📁 Project Structure

```
src/
├── components/
│   ├── Header.jsx       # Navigation header
│   ├── Hero.jsx         # Hero section
│   ├── About.jsx        # About section
│   ├── Skills.jsx       # Skills section
│   ├── Projects.jsx     # Projects showcase
│   ├── Contact.jsx      # Contact form
│   └── Footer.jsx       # Footer
├── App.jsx              # Main app component
├── main.jsx            # Entry point
└── index.css           # Global styles
```

## 🚀 Getting Started

### Prerequisites

- Node.js (v18 or higher)
- npm or yarn

### Installation

1. Clone the repository
```bash
git clone https://github.com/dineshsk27/dinesh-karuppasamy.github.io.git
cd dinesh-karuppasamy.github.io
```

2. Install dependencies
```bash
npm install
```

3. Start the development server
```bash
npm run dev
```

4. Open [http://localhost:5173](http://localhost:5173) to view it in the browser.

## 🎨 Customization

### Personal Information
Update the following files with your information:
- `src/components/Hero.jsx` - Name, title, and introduction
- `src/components/About.jsx` - About section content
- `src/components/Contact.jsx` - Contact information
- `src/components/Footer.jsx` - Footer details

### Projects
Edit `src/components/Projects.jsx` to showcase your projects:
- Add your project details
- Update GitHub and live demo URLs
- Replace placeholder images

### Skills
Modify `src/components/Skills.jsx` to reflect your skills:
- Update skill categories
- Adjust skill levels
- Add or remove technologies

### Styling
- Colors can be customized in `tailwind.config.js`
- Global styles are in `src/index.css`

## 🚀 Deployment

### GitHub Pages

1. Build the project
```bash
npm run build
```

2. Deploy to GitHub Pages
```bash
npm run deploy
```

The site will be available at `https://dineshsk27.github.io/dinesh-karuppasamy.github.io/`

### Manual Deployment

1. Build the project
```bash
npm run build
```

2. The `dist` folder contains the built files ready for deployment
